# Multimodal Video Compliance Auditor

An AI-powered application for automated video advertisement compliance analysis. The system combines video understanding, OCR, transcript analysis, and policy retrieval to evaluate whether a video satisfies regulatory guidelines and produces a structured compliance report.

## Tech Stack

* Python
* FastAPI
* LangGraph
* Azure OpenAI
* Azure Video Indexer
* Vector Search
* Next.js
* TypeScript

## Features

* Automated video analysis
* OCR and transcript processing
* Policy-aware compliance evaluation
* Agentic workflow using LangGraph
* Interactive web interface for submitting and reviewing analyses

> This project is under active development.
